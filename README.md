## Install

```sh
- npm install axios
- npm i react-axios
- npm install moment --save
```
## Tools & API
- [Bootstrap](https://getbootstrap.com/)
- [Fontaweosome](https://fontawesome.com/)
- [Openweather](https://openweathermap.org/)
- [Unsplash](https://unsplash.com/)


## Usage
```sh
npm run start
```
